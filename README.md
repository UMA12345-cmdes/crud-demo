# Flutter Notes App

## Tech Stack
- Flutter
- Firebase Authentication
- Cloud Firestore

## Features
- Email & password authentication
- Persistent login
- Create, edit, delete notes
- User-specific secure data access
- Search notes by title

## Setup Instructions
1. Clone the repository
2. Run `flutter pub get`
3. Run `flutter run`

## Firebase Configuration
- Collection: noteList
- Fields:
  - id
  - title
  - content
  - created_at
  - updated_at
  - user_id
